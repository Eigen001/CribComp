# CribComp — Claude Code Instructions

## Project
This repo contains a single-file HTML5 app: `index.html`
It is a Cribbage score tracking companion app.

## Workflow for updating the app
When asked to update the app or push a new version:
1. Copy the new index.html content into `index.html` in this repo
2. Run: git add index.html
3. Run: git commit -m "Update CribbageCompanion - [brief description of change]"
4. Run: git push origin main

## File structure
- index.html — the entire app (single file, all HTML/CSS/JS inline)
- CLAUDE.md — this file

## Notes
- Never modify any other files unless explicitly asked
- Always push to the `main` branch
- Commit message should briefly describe what changed
